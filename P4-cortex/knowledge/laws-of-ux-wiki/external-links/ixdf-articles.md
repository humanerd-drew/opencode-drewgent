## Fitts' Law

### [Fitts' Law - Topic Overview](https: //www.interaction-design.org/literature/topics/fitts-law)

**Definition: ** The amount of time required for a person to move a pointer to a target area is a function of distance divided by target size.

**Key Points: **
- Time to acquire target = f(distance / size)
- Longer distance + smaller target = longer movement time
- Applicable to rapid, pointing movements (not continuous motion)
- Influenced convention of making interactive buttons large

**UI Applications: **
- Large touch targets for mobile devices
- Keep distance between user's focus area and related buttons short
- Right-click pop-up menus minimize travel distance
- Screen edges = "magic pixels" (infinite targets)

### [Fitts's Law: The Importance of Size and Distance in UI Design](https://www.interaction-design.org/literature/article/fitts-s-law-the-importance-of-size-and-distance-in-ui-design)

**Core Formula: ** T (Time) = a + b log2(2D/W)

- T = movement time
- a, b = regression coefficients
- D = distance to target
- W = width of target

**UI Design Applications: **
1. Command buttons must be distinguished by size (larger = easier to click)
2. Screen edges/corners acquired faster (pinned by screen boundary)
3. Pop-up menus better than dropdowns (no cursor travel needed)
4. Pie menus equal distance to all options
5. Task bars impede movement

**Magic Pixels: ** Screen corners are farthest from prime pixel (center) - least valuable real estate for important actions.
