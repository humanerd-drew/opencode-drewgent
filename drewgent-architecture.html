Drewgent System Architecture

AI-augmented personal operating system — P-layer brain + Kanban pipeline + Content pipeline

Components

  • • 5 MCP servers (discord, wordpress, gjc, codebase-memory, portone)
  • • 9 launchd services with KeepAlive
  • • 12 cron jobs dispatched via drewgent_cron.py
  • • 15 agent profiles (explorer → implementer → reviewer)
  • • P-Layer vault: 7 Obsidian directories (P0-P6)

Tech Stack

  • • Runtime: opencode (DeepSeek V4 + Kimi K2.7 + Qwen 3.7)
  • • Storage: SQLite (knowledge.db + kanban) + Obsidian FS
  • • Infra: launchd (macOS) + colima (Docker) + Cloudflare Tunnel
  • • Languages: Python 3.14, Node.js, shell
  • • Frontend: WordPress + GeneratePress (YOUR_DOMAIN)

Key Patterns

  • • Tiered Autonomy: T1 autonomous → T4 human-only decision
  • • Answer-First Comm: conclusion before detail
  • • Ponytail: YAGNI + stdlib + one-liner first
  • • Kanban Pipeline: task → explorer → implementer → reviewer
  • • Provenance: every artifact carries trigger + decision context
